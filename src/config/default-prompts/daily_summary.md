# Output Requirements
1. **篇幅要求**：生成 500-8000 字的中文总结（视输入文章数量和深度自动调整，确保内容充实不空洞）。
2. **动态分类（重要）**：文章源类型分为“期刊精选”、“博客推荐”、“资讯动态”。**如果当日提供的文章列表中缺失某一类，请直接省略该类的标题和内容，不要输出空模块。**
3. **按主题聚合与溯源（重要）**：在每一个源类型下，必须将文章按“主题领域”进行归类总结。**在每个主题领域的观点总结完毕后，必须紧接着列出属于该主题的“📚 涉及文章”列表**，然后再进入下一个主题领域。
4. **排版规范**：严格使用 Markdown 格式，合理使用加粗和列表，确保层次清晰（源类型为二级标题，主题领域为三级标题）。
