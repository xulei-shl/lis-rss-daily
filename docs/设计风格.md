# LIS-RSS-Daily UI 设计规范指南 (Design System)

本文档旨在确立网站的核心设计风格，基于 `filters.css` 的重构经验，为后续页面元素的优化提供统一标准。

## 1. 核心设计理念 (Core Philosophy)

*   **高级简约 (Premium Minimalism)**: 摒弃繁杂装饰，利用空间、留白和微妙的阴影来构建层次感。
*   **几何对齐 (Geometric Alignment)**: 强调视觉元素的严格对齐。利用 Grid 系统确保垂直方向上的视觉律动 (Vertical Rhythm)。
*   **触感交互 (Tactile Interaction)**: 控件在交互（Hover, Focus）时应提供平滑、有质感的反馈。

---

## 2. 布局与容器 (Layout & Containers)

### 面板容器 (Panel)
所有浮动的卡片、筛选栏、内容块应遵循以下容器样式：

*   **背景**: `var(--bg-elevated)` (通常比背景稍亮或从属色)
*   **边框**: `1px solid var(--border)`
*   **圆角**: `12px` (大圆角，柔和亲切)
*   **阴影**: `0 4px 20px -4px rgba(0, 0, 0, 0.08)` (弥散感强的高级阴影，非生硬的黑边)
*   **内边距**: `24px` (舒适的呼吸感)

### 栅格系统 (Grid System)
在处理多列复杂表单或布局时，**优先使用 CSS Grid** 而非 Flexbox，以保证跨行元素的严格对齐。

> **示例策略**:
> *   行1: `1fr 2fr` (1:2 比例)
> *   行2: `1fr 1fr 1fr 3fr` (1:1:1:3 比例)
> *   *对齐魔法*: 行1的 `1fr` 视觉宽度 === 行2的 `1fr + 1fr`，创造潜意识的秩序美。

---

## 3. 组件规范 (Component Specifications)

### 通用输入控件 (Inputs, Selects, Date Pickers)
所有交互控件必须严格遵守以下物理属性，禁止出现高度参差不齐的情况。

| 属性 | 规范值 | 说明 |
| :--- | :--- | :--- |
| **高度 (Height)** | **42px** | **强制统一**。这是整个 UI 的垂直基准。 |
| **圆角 (Radius)** | **8px** | 现代标准圆角。 |
| **边框 (Border)** | `1px solid var(--border)` | 默认状态。 |
| **背景 (Background)** | `var(--bg-surface)` | |
| **字体 (Font)** | `14px` (text-sm), Weight 500, `var(--font-body)` | 清晰，字重适中。 |
| **盒模型** | `border-box` | 内边距不撑大尺寸。 |
| **配色方案** | `color-scheme: light dark;` | **关键**：强制原生弹窗（如日历）适配暗黑模式。 |

### 状态交互 (Interaction States)

*   **Hover**: 边框颜色变为 `var(--text-tertiary)` 或更深，提供悬停提示。
*   **Focus**:
    *   `outline: none`
    *   `border-color: var(--accent-primary)` (主色调)
    *   `box-shadow: 0 0 0 3px var(--accent-subtle)` (3px 扩散光晕，不要用默认的 blue outline)
    *   `z-index: 10` (如需，防止边框被遮挡)
*   **Transition**: `all 0.2s cubic-bezier(0.4, 0, 0.2, 1)` (丝般顺滑的过渡)。

---

## 4. 图标与细节 (Icons & Details)

### 图标着色 (Iconography)
*   **原则**: 严禁使用浏览器默认的黑色箭头或日历图标。
*   **实现**: 使用 SVG (`background-image` 或 `mask`)。
*   **颜色**: 统一使用中性灰 `#9CA3AF` 或 `var(--text-tertiary)`。这能确保在亮色和暗色模式下都有良好的可见性，且不刺眼。
*   **位置**:
    *   搜索图标：左偏 `14px`。
    *   下拉箭头：右偏 `10px`。

### 间距控制 (Spacing)
*   **搜索框文本**: `padding-left: 42px` (为图标预留足够空间，避免重叠)。
*   **下拉框文本**: `padding-right: 36px` (防止文本遮挡箭头)。

---

## 5. 通用 CSS 代码片段 (Snippets)

复制此代码块以快速应用新风格：

```css
/* 容器风格 */
.panel-style {
  background: var(--bg-elevated);
  border: 1px solid var(--border);
  border-radius: 12px;
  box-shadow: 0 4px 20px -4px rgba(0, 0, 0, 0.08); 
  padding: 24px;
}

/* 输入框核心风格 */
.input-style {
  height: 42px; /* 核心 */
  width: 100%;
  padding: 0 12px;
  border-radius: 8px;
  border: 1px solid var(--border);
  background: var(--bg-surface);
  color: var(--text-primary);
  font-size: 14px;
  font-weight: 500;
  appearance: none;
  box-sizing: border-box;
  color-scheme: light dark; /* 暗黑适配 */
  transition: all 0.2s ease;
}

/* 输入框悬停/聚焦 */
.input-style:hover {
  border-color: var(--text-tertiary);
}
.input-style:focus {
  outline: none;
  border-color: var(--accent-primary);
  box-shadow: 0 0 0 3px var(--accent-subtle);
}

/* 自定义下拉箭头 (示例) */
.select-style {
  padding-right: 36px;
  background-image: url("data:image/svg+xml,..."); /* 替换为灰色 SVG */
  background-position: right 10px center;
  background-repeat: no-repeat;
}
```
