# 深度检索输入模板

## 格式说明

每行一篇种子文献，格式如下：

### 方式一：有 ID（推荐）
```
- 文献标题@ID
```

### 方式二：有 ID（兼容旧格式）
```
- 文献标题：ID
- 文献标题:ID
```

### 方式三：无 ID
```
- 文献标题
```

## 示例

```
- 从替代到协同：AIGC赋能图书馆人智协同咨询服务模式研究@405
- 深度学习在医学影像中的应用：12345
- 基于Transformer的图像分割方法
- 多模态大语言模型研究进展
- 医学图像配准与融合技术
- 计算机辅助诊断系统综述
```

## 参数说明

- **迭代轮次 (rounds)**：0-3 轮，轮次越多搜索越深入，但耗时更长
- **相关文献数量 (semantic_limit)**：每轮检索的文献数量
- **最大保留文章数 (max_final_articles)**：最终保留的最大文章数 (5-50)
- **相关性阈值 (score_threshold)**：相似度阈值，越高相关性要求越严格

## 注意事项

1. 请确保每行以 `- ` 开头
2. ID 通常是文献的唯一标识符（如 DOI、PMID 等）
3. 题名中若包含冒号，建议使用 `@ID` 分隔题名和 ID
4. 输入内容将作为种子进行语义搜索和扩展
